# Подсказка по гиту

## Инициализация репозитория

```sh
git init
```

## Статус репозитория

```sh
git status
```

## Сохранить изменения

```sh
git add 
```

## Сделать коммит

```sh
git commit -m
```

## Показывает в какой ветке веработаешь

```sh
git branch
```

## Создать новую ветку

```sh
git branch branch_mame
```

## Переключение между ветками

```sh
git checkout
```
